# Windows AppsFolder 打包应用发现与启动 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在保留 Start Menu `.lnk` 扫描的同时，原生发现并启动 Windows AppsFolder 中具有有效 packaged AUMID 的应用，并通过入口类型副标题区分同名结果。

**Architecture:** `apps::discovery` 继续负责一次完整 refresh 的编排，新 `apps::appsfolder` 只封装 AppsFolder Shell 枚举、字符串所有权与 AUMID 过滤。`Application` 和 Rust-only `ResultAction` 共享一个可信启动目标枚举；Windows backend 分别执行桌面快捷方式策略和 `IApplicationActivationManager` 启动。前端继续消费现有 `ResultItem.subtitle`，不改协议或组件。

**Tech Stack:** Rust 1.77.2、Tauri 2、`windows` crate 0.61.3、Windows CNG SHA-256、Windows Shell/COM、现有 Rust 单元测试、Vitest/TypeScript 构建。

**Source Spec:** `docs/superpowers/specs/2026-07-21-windows-appsfolder-discovery-design.md`

## Global Constraints

- 实现基线固定为 `bb4a03c48ee0bb5b03b5b69294a54bb3bc76cd24`；只在 `codex/windows-appsfolder-app-discovery` 工作。
- 保留 Start Menu `.lnk` 扫描；AppsFolder 只接受 `ParseApplicationUserModelId` 验证成功的 packaged AUMID。
- 使用 `windows = 0.61.3`；只增加必要 feature，不增加 crate、PowerShell 子进程、注册表或包清单 fallback。
- AUMID、快捷方式路径、可执行路径与 `ApplicationLaunchTarget` 只存在于 Rust，不序列化给前端。
- 单项坏数据只增加定长、无路径诊断；AppsFolder 绑定或枚举失败使完整 refresh 失败并保留旧缓存。
- 排序保留 match/alias/use_count 语义；只有这些条件及名称相同时，`PackagedApp` 才优先于 `DesktopShortcut`。
- 每项生产行为必须先运行最小 RED，再写 GREEN；不使用 sleep、不启动真实应用作为自动测试。
- 每次 `cargo test` 后检查 8 个 `permissions/autogenerated` 文件；若 raw/hash 无内容差异，只用 `git add -- <paths>` 刷新 stat，并验证 cached diff 为空。

```rust
#[derive(Clone, Copy, Debug, Eq, PartialEq)]
pub(crate) enum ApplicationEntryKind {
    DesktopShortcut,
    PackagedApp,
}

#[derive(Clone, Debug, Eq, PartialEq)]
pub(crate) enum ApplicationLaunchTarget {
    Shortcut {
        shortcut: PathBuf,
        executable: Option<PathBuf>,
    },
    PackagedApp {
        aumid: String,
    },
}

impl ApplicationLaunchTarget {
    pub(crate) fn entry_kind(&self) -> ApplicationEntryKind {
        match self {
            Self::Shortcut { .. } => ApplicationEntryKind::DesktopShortcut,
            Self::PackagedApp { .. } => ApplicationEntryKind::PackagedApp,
        }
    }
}
```
